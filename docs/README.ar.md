## الميزات

- 🧠 **Gemma 4 E2B** — دماغ ذكاء اصطناعي متعدد الوسائط (نص + صوت + رؤية)، أكثر من 140 لغة
- 🎤 **إدخال صوتي** — اضغط للتحدث مع عرض موجة أوسيلوسكوب مباشرة
- 🔊 **إخراج صوتي** — تركيب كلام متعدد اللغات MMS-TTS (إنجليزي، تايلاندي، إلخ)
- 👁️ **رؤية** — تحليل الصور في نظام الملفات عبر مشفر الرؤية في Gemma
- 🔧 **أدوات وكيلة** — فتح ملفات، قراءة ملفات، عرض المجلدات، تنفيذ أوامر
- ⚠️ **موافقة أمنية** — كل إجراء أداة يتطلب موافقة صريحة من المستخدم عبر نافذة منبثقة
- 🖥️ **واجهة طرفية رجعية** — طرفية خضراء نيون مع تسلسل إقلاع، مدعومة بـ Ratatui
- 🔒 **بدون إنترنت 100%** — بدون سحابة، بدون مفاتيح API، لا تغادر البيانات جهازك
- 🍎 **أصلي لنظام Mac** — محسّن لـ Apple Silicon (M1+)

## البنية

```
terminator (ملف Rust ثنائي)
├── ratatui       → واجهة طرفية CRT رجعية
├── cpal          → التقاط الميكروفون (PCM 16kHz أحادي)
└── جسر Python (subprocess، بروتوكول JSON)
    ├── HuggingFace Transformers
    │   └── Gemma 4 E2B (نص + صوت + رؤية، نموذج واحد)
    └── MMS-TTS (facebook/mms-tts-eng، mms-tts-tha، إلخ)
```


## المتطلبات

- macOS على Apple Silicon (M1+)
- Rust 1.75+
- Python 3.11+
- ~5 جيجابايت قرص (أوزان النموذج)
- ~4 جيجابايت ذاكرة (الاستدلال)

## البدء السريع

```bash
# 1. استنساخ
git clone https://github.com/zixma13/terminator.git
cd terminator

# 2. تثبيت تبعيات Python
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. تنزيل النموذج (المرة الأولى فقط، ~5 جيجابايت)
python3 scripts/download_model.py

# 4. البناء والتشغيل
cargo build --release
./target/release/terminator
```

## الأدوات (وكيلة)

يمكن لـ TERMINATOR التفاعل مع نظامك عبر استدعاءات الأدوات. كل إجراء يتطلب موافقة صريحة عبر نافذة تحذير منبثقة.

| الأداة            | الوصف                                |
|-------------------|--------------------------------------|
| `open_file`       | فتح ملف/مجلد في التطبيق الافتراضي    |
| `read_file`       | قراءة وإرجاع محتوى ملف نصي          |
| `list_directory`  | عرض الملفات في مجلد                  |
| `run_command`     | تنفيذ أمر shell                      |
| `analyze_image`   | تحليل صورة باستخدام Gemma 4 vision   |

مثال: *"اعرض الملفات في التنزيلات"* → نافذة موافقة منبثقة → تنفيذ `ls` → الذكاء الاصطناعي يلخص النتائج.
