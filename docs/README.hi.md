## विशेषताएँ

- 🧠 **Gemma 4 E2B** — मल्टीमोडल AI ब्रेन (टेक्स्ट + ऑडियो + विज़न), 140+ भाषाएँ
- 🎤 **वॉइस इनपुट** — पुश-टू-टॉक, लाइव ऑसिलोस्कोप वेवफ़ॉर्म के साथ
- 🔊 **वॉइस आउटपुट** — MMS-TTS बहुभाषी स्पीच सिंथेसिस (अंग्रेज़ी, थाई आदि)
- 👁️ **विज़न** — Gemma विज़न एनकोडर से फ़ाइल सिस्टम की छवियों का विश्लेषण
- 🔧 **एजेंटिक टूल्स** — फ़ाइलें खोलें, पढ़ें, डायरेक्टरी सूची, कमांड चलाएँ
- ⚠️ **सुरक्षा अनुमोदन** — हर टूल एक्शन के लिए पॉपअप द्वारा स्पष्ट अनुमति आवश्यक
- 🖥️ **रेट्रो CRT UI** — बूट सीक्वेंस के साथ नियॉन ग्रीन टर्मिनल, Ratatui द्वारा संचालित
- 🔒 **100% ऑफ़लाइन** — कोई क्लाउड नहीं, कोई API कुंजी नहीं, डेटा आपकी मशीन से बाहर नहीं जाता
- 🍎 **Mac नेटिव** — Apple Silicon (M1+) के लिए अनुकूलित

## आर्किटेक्चर

```
terminator (Rust बाइनरी)
├── ratatui       → रेट्रो CRT टर्मिनल UI
├── cpal          → माइक्रोफ़ोन कैप्चर (PCM 16kHz mono)
└── Python ब्रिज (subprocess, JSON प्रोटोकॉल)
    ├── HuggingFace Transformers
    │   └── Gemma 4 E2B (टेक्स्ट + ऑडियो + विज़न, 1 मॉडल)
    └── MMS-TTS (facebook/mms-tts-eng, mms-tts-tha आदि)
```

### डेटा फ़्लो

```
🎤 वॉइस → cpal → Gemma 4 (ट्रांसक्राइब) → Gemma 4 (सोच/टूल्स) → MMS-TTS 🔊
⌨️ टेक्स्ट →                                Gemma 4 (सोच/टूल्स) → MMS-TTS 🔊
                                              ↓
                                    [टूल कॉल डिटेक्ट]
                                              ↓
                                    ⚠ चेतावनी पॉपअप [Y/N]
                                              ↓
                                    निष्पादन (खोलें/पढ़ें/सूची/चलाएँ/विज़न)
                                              ↓
                                    परिणाम → Gemma 4 → प्रतिक्रिया
```

## आवश्यकताएँ

- Apple Silicon (M1+) पर macOS
- Rust 1.75+
- Python 3.11+
- ~5GB डिस्क (मॉडल वेट)
- ~4GB RAM (इन्फ़रेंस)

## त्वरित शुरुआत

```bash
# 1. क्लोन
git clone https://github.com/zixma13/terminator.git
cd terminator

# 2. Python डिपेंडेंसी इंस्टॉल करें
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. मॉडल डाउनलोड करें (केवल पहली बार, ~5GB)
python3 scripts/download_model.py

# 4. बिल्ड और रन
cargo build --release
./target/release/terminator
```

## टूल्स (एजेंटिक)

TERMINATOR टूल कॉलिंग के ज़रिए आपके सिस्टम से इंटरैक्ट कर सकता है। हर एक्शन के लिए चेतावनी पॉपअप द्वारा स्पष्ट अनुमोदन आवश्यक है।

| टूल               | विवरण                                |
|-------------------|--------------------------------------|
| `open_file`       | डिफ़ॉल्ट ऐप में फ़ाइल/फ़ोल्डर खोलें    |
| `read_file`       | टेक्स्ट फ़ाइल की सामग्री पढ़ें और लौटाएँ |
| `list_directory`  | डायरेक्टरी में फ़ाइलों की सूची दिखाएँ   |
| `run_command`     | शेल कमांड निष्पादित करें              |
| `analyze_image`   | Gemma 4 विज़न से छवि का विश्लेषण करें |

उदाहरण: *"मेरे डाउनलोड्स में फ़ाइलें दिखाओ"* → अनुमोदन पॉपअप → `ls` निष्पादित → AI परिणाम सारांशित करता है।

## संसाधन उपयोग

Apple M5 Pro (48GB) पर मापा गया:

| कंपोनेंट          | CPU     | RAM      |
