## Возможности

- 🧠 **Gemma 4 E2B** — мультимодальный ИИ-мозг (текст + аудио + зрение), 140+ языков
- 🎤 **Голосовой ввод** — нажми и говори с отображением осциллограммы в реальном времени
- 🔊 **Голосовой вывод** — многоязычный синтез речи MMS-TTS (английский, тайский и др.)
- 👁️ **Зрение** — анализ изображений в файловой системе через визуальный кодировщик Gemma
- 🔧 **Агентные инструменты** — открытие файлов, чтение, список каталогов, выполнение команд
- ⚠️ **Подтверждение безопасности** — каждое действие инструмента требует явного одобрения пользователя через всплывающее окно
- 🖥️ **Ретро CRT UI** — неоново-зелёный терминал с последовательностью загрузки, на базе Ratatui
- 🔒 **100% офлайн** — без облака, без API-ключей, данные не покидают вашу машину
- 🍎 **Нативный для Mac** — оптимизирован для Apple Silicon (M1+)

## Архитектура

```
terminator (бинарный файл Rust)
├── ratatui       → ретро CRT-терминальный UI
├── cpal          → захват микрофона (PCM 16кГц моно)
└── Python-мост (subprocess, JSON-протокол)
    ├── HuggingFace Transformers
    │   └── Gemma 4 E2B (текст + аудио + зрение, 1 модель)
    └── MMS-TTS (facebook/mms-tts-eng, mms-tts-tha и др.)
```

### Поток данных

```
🎤 Голос → cpal → Gemma 4 (транскрипция) → Gemma 4 (мышление/инструменты) → MMS-TTS 🔊
⌨️ Текст →                                 Gemma 4 (мышление/инструменты) → MMS-TTS 🔊
                                              ↓
                                    [обнаружен вызов инструмента]
                                              ↓
                                    ⚠ Всплывающее ПРЕДУПРЕЖДЕНИЕ [Y/N]
                                              ↓
                                    Выполнить (открыть/прочитать/список/выполнить/зрение)
                                              ↓
                                    Результат → Gemma 4 → ответ
```

## Требования

- macOS на Apple Silicon (M1+)
- Rust 1.75+
- Python 3.11+
- ~5 ГБ диска (веса модели)
- ~4 ГБ ОЗУ (инференс)

## Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/zixma13/terminator.git
cd terminator

# 2. Установить зависимости Python
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# 3. Скачать модель (только при первом запуске, ~5 ГБ)
python3 scripts/download_model.py

# 4. Собрать и запустить
cargo build --release
./target/release/terminator
```

## Управление

| Клавиша   | Действие                              |
|-----------|---------------------------------------|
| `Enter`   | Отправить набранное сообщение         |
| `Tab`     | Переключить режим голос/текст         |
| `Space`   | Нажать для начала/остановки записи (голос) |
| `Y`       | Одобрить действие инструмента         |
| `N`       | Отклонить действие инструмента        |
| `Esc`     | Выход                                 |

## Инструменты (Агентные)

TERMINATOR может взаимодействовать с вашей системой через вызовы инструментов. Каждое действие требует явного одобрения через всплывающее предупреждение.

| Инструмент        | Описание                             |
|-------------------|--------------------------------------|
| `open_file`       | Открыть файл/папку в приложении по умолчанию |
| `read_file`       | Прочитать и вернуть содержимое текстового файла |
| `list_directory`  | Показать список файлов в каталоге    |
| `run_command`     | Выполнить команду оболочки           |
| `analyze_image`   | Анализировать изображение с Gemma 4 vision |

Пример: *«Показать файлы в моих Загрузках»* → всплывающее одобрение → выполняет `ls` → ИИ резюмирует результаты.

## Использование ресурсов

Измерено на Apple M5 Pro (48 ГБ):

| Компонент         | CPU     | ОЗУ      |
|-------------------|---------|----------|
| Rust TUI          | < 1%    | ~22 МБ   |
| Python/Gemma 4    | ~78%*   | ~4,2 ГБ  |
| MMS-TTS           | burst   | ~145 МБ  |
| **Итого**         |         | **~4,3 ГБ** |

*\*CPU нагружается только во время инференса, в остальное время простаивает. GPU (Metal) используется для ускорения.*
